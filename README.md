Media Explorer App

## Descripción

Media Explorer App es una aplicación desarrollada en Flutter que permite explorar productos obtenidos desde una API pública, visualizar sus detalles y administrar una colección personalizada mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

La aplicación implementa navegación entre pantallas, consumo de servicios REST, almacenamiento temporal en memoria y scroll infinito para mejorar la experiencia del usuario.

---

## Características

* Consumo de API REST utilizando HTTP.
* Listado dinámico de productos.
* Visualización de detalles de cada producto.
* Scroll infinito para cargar más elementos.
* Agregar productos a una colección personal.
* Editar productos guardados.
* Eliminar productos de la colección.
* Navegación entre múltiples pantallas.
* Interfaz desarrollada con Flutter Material Design.

---

## Tecnologías Utilizadas

* Flutter
* Dart
* HTTP Package
* Mongo Dart (configurado para futuras integraciones)
* UUID

## Funcionalidades Implementadas

### Exploración de API

La aplicación consume datos desde una API pública y muestra información relevante de cada producto.

### Scroll Infinito

Los productos se cargan progresivamente conforme el usuario navega por la lista.

### CRUD Local

* Crear productos.
* Leer productos almacenados.
* Actualizar información de productos.
* Eliminar productos.

### Gestión de Colección

Los productos seleccionados pueden guardarse en una colección personalizada para su posterior administración.
